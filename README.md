# Engineering Docs Toolkit

Reusable build tooling for engineering books, workshop manuals, translated technical references, and specification documents.

The toolkit provides one common publication engine for projects such as HERKULES 1934, Atarix documentation, Catalyst specifications, and future technical translation projects.

## Goals

- Build Markdown source into HTML, DOCX, EPUB, and PDF-oriented outputs.
- Generate glossaries, indexes, concordances, and reference appendices.
- Support translation workflows with source notes and target-language chapters.
- Keep publication logic reusable across many repositories.

Initial scaffold in progress.
