# Roadmap

## 0.1.0 Foundation

- CLI with init, build, and check commands.
- book.yaml project configuration.
- Markdown assembly into book.md and book.html.
- First-pass MathML rendering.
- Glossary and index summary generation.
- Minimal accessibility checks.
- CI tests and minimal example build.

## Later

- Semantic MathML conversion.
- OCR and PDF import pipeline.
- EPUB accessibility metadata.
- DOCX, EPUB, and PDF exporters.
- Plugin architecture.
- Translation memory.
